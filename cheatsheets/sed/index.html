Front Hotkey Command Notes Perskey
Delete all lines not matching pattern sed ‘/pattern/!d’ Try to run with single quotes in bash as !d might be interpreted
Replace old word with new sed ’s/old/new/g' Without g, it only replaces the first one in each line
How to run multiple expressions? sed -e ’expr’ -e ’expr'
Only delete if line matches pattern sed ‘/pattern/d’ Note: this /pattern part can be applied to the start of other commands
What if / is part of your pattern? You can use other symbols such as # or | to separate parts of the sed expression
How to reference the matched string in a substitution? sed ’s/old/&/g'
Substitution, change case sed ’s/old/\U&/g' L for lowercaseLowercase l&u only act on one letter
Use extended regex sed -E Supports capture groups, numbers, etc. No need to escape \1
Ignore case in a sed pattern? Add I to the end (same place as gc)
Match repeating content /\([a-z]\)\1/
