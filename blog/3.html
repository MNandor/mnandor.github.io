Vim Cheatsheet

2022-06-15

My personal Vim cheat sheet, information collected from various sources. This post was exported from Markdown and is compatible with my Joplin-to-Anki program.

There might be a couple inaccuracies, I just wanted to see how this looked as a website.

Motions

  • motions, when alone, move the cursor.
    • l moves the cursor right →
  • prepend a number to the motion to repeat n times.
    • 5l moves 5 characters to the right →
  • some commands expect a motion.
    • dl deletes 1 character to the right →
    • d5l deletes 5 characters to the right →
Front Hotkey Command Notes Personal Hotkey

J looks like a down arrow.
Jump to line 5 5gg
5G		 :5
Jump to the start and end of a line 0 ^ $ Same as regex symbols.
^ jumps to first non-whitespace.
Last non-whitespace is g_
Jump to top, bottom of file gg
G
Jump to other end of parentheses % When not touching one, jumps to last open parenthesis.

Word Motions

  • motions between words, separated by spaces
  • without capitals, punctuation is treated as a separate word
Front Hotkey Command Notes Personal Hotkey
Motion until to the next word-start w W W considers punctuation to be part of the word.
Motion until the next word-end e E E considers punctuation to be part of the word.
Motion until the last word-start b B B considers punctuation to be part of the word.
Until the last word-end ge gE gE considers punctuation to be part of the word.

Text Objects

  • act as the selection for commands
  • i (inside) / a (around): whether to include the boundary (such as spaces or parentheses)
  • commands are included in the examples, but different commands work with the same text objects
